プロンプトジェネレーター

リストから単語を選ぶだけでno〇elAIなどで使用できるプロンプトを生成できます。

またモードをサイトモードにすればru〇e34やe6〇1で検索用のタグも生成できます。

単語の登録もできますがサイトに反映することができない(静的動作)ためお手数ですが[JSON保存]でデータを保存、自分に[prompt_words.json]を送ってください。

〇一括登録機能について

行ごとに

[英語単語]=[日本語単語]

と記載したテキストファイルをテキストインポートで読み込ませれば一括で単語を登録できます。

もし、不具合などありましたら自分のDMに連絡してください。
